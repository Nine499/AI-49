# delnonwebp.ps1 使用说明

## 功能简介

本脚本用于清理指定目录下的非`.webp`格式文件及产生的空文件夹，支持递归操作。

## 脚本特性

1. **精准删除**

   - 仅保留`.webp`格式文件（不区分大小写）
   - 支持处理带隐藏属性/系统属性的文件
   - 通过`-LiteralPath`参数确保文件路径中的特殊字符安全处理

2. **智能清理空文件夹**
   - 按文件夹深度倒序删除（先删子目录，后删父目录）
   - 空文件夹判定标准：目录中完全不含任何文件（含隐藏文件）

## 使用步骤

1. 修改目标路径：
   ```powershell
   $targetDir = "C:\Users\fortynine\acg\1"  # ← 修改为你的实际路径
   ```
2. 执行脚本：
   ```powershell
   .\delnonwebp.ps1
   ```

## 注意事项

⚠️ **危险操作预警**

- 删除操作不可逆，请务必提前备份重要数据
- 建议首次运行前在测试目录验证效果

💡 推荐增强措施：

1. 添加日志记录功能（通过`Start-Transcript`）
2. 在删除前添加确认提示（`-WhatIf`参数测试）
3. 设置文件保留白名单（扩展名数组）

## 常见问题

Q：为什么需要按文件夹深度倒序删除？
A：确保先删除子目录的空文件夹，防止父目录删除后子目录无法访问

Q：如何查看被删除的文件列表？
A：可添加`-Verbose`参数到`Remove-Item`命令
