# yasuo.ps1 使用文档

## 项目简介

yasuo.ps1 是一个 PowerShell 自动化压缩工具，用于批量压缩指定目录下的所有子文件夹。通过调用 7-Zip 实现自动化归档操作，适用于文件夹集合备份、日志归档等场景。

## 系统依赖

- Windows 操作系统（PowerShell 5.0+）
- 已安装 [7-Zip](https://www.7-zip.org/) 压缩工具
- 7-Zip 路径已添加到系统环境变量（可通过 `7z` 命令验证）

## 使用方法

```powershell
# 执行示例
.\yasuo.ps1 -targetPath "C:\目标文件夹路径"
```

## 工作流程

1. **参数验证**

   - 必须通过 `-targetPath` 指定目标路径
   - 自动检测路径有效性（路径不存在时终止执行）

2. **目录处理**

   - 遍历目标路径下的所有直接子文件夹
   - 为每个子文件夹创建同名 `.7z` 压缩包

3. **压缩执行**
   - 使用 `7z a` 命令进行压缩
   - 自动添加 `-y` 参数覆盖已有文件
   - 压缩包生成在原目标文件夹内

## 输出示例

假设目录结构：

```
D:\测试目录\
├─文档1
├─文档2
└─图片集
```

执行命令后：

```
D:\测试目录\
├─文档1.7z      ← 压缩包
├─文档2.7z      ← 压缩包
├─图片集.7z     ← 压缩包
├─文档1         ← 原始文件夹
├─文档2         ← 原始文件夹
└─图片集        ← 原始文件夹
```

## 注意事项

1. **执行策略**

   - 首次运行需设置执行权限：
     ```powershell
     Set-ExecutionPolicy RemoteSigned
     ```

2. **文件覆盖**

   - 同名压缩包会被自动覆盖（`-y` 参数作用）

3. **错误处理**

   - 路径不存在时立即终止
   - 压缩失败时显示黄色警告
   - 建议以管理员权限运行确保文件访问权限

4. **性能建议**
   - 不建议处理包含大量小文件的文件夹
   - 大文件压缩建议使用 `-mx=0` 参数优化速度

## 常见问题

**Q：为什么压缩后的文件夹依然存在？**
A：该脚本仅生成压缩包，不会删除原始文件夹。如需清理请手动删除或扩展脚本功能。

**Q：如何修改压缩格式？**
A：修改 `$command` 变量中的 `a` 参数为目标格式（如 `a -tzip` 生成 ZIP 格式）。

**Q：出现"7z 不是内部命令"错误？**
A：请确认 7-Zip 已正确安装并添加到系统环境变量 path 中。
